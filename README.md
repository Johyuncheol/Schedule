●프로젝트 소개


캠프 남은 기간 동안 여러 번의 팀 프로젝트를 계속 진행 할텐데

서로 할 일들과 진행 상황을 공유하면서 할 일 -진행 중-완료 루트로 시각적으로

한눈에 알아 보기 편한 서비스가 있으면 좋겠다는 생각이 들었고

그러면 '직접 만들어서 사용해보자!' 라는 마음에서 시작했습니다.
